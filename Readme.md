## Marshall rule

### intro
We study, in this project, how neural networks capturing known phyiscal rules.

### usage

Run training 
```
python main.py --is_train=True 
```

There are following options
* is_train
* task_name
* DATA_PATH
* LABEL_PATH
* NUM_EPOCH
* BATCH_SIZE

For testing
```
python main.py --is_train=False
```
